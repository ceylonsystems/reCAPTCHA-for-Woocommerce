Adds a reCAPTCHA verification for your default Woocommerce registration form in My Account page.

== Description ==

This plugin is adding a reCAPTCHA verification for the registration process in the My Account page of Woocommerce.

== Installation ==

= Automatic Installation =

1. In the dashboard navigate to `Plugins > Add New`
2. Search for `Woocommerce reCAPTCHA`
3. Locate the `Woocommerce reCAPTCHA` by `Ceylon Systems` and click `Install Now`
4. After installation, click on `Activate Plugin` to activate

= Manual Installation =

1. Download the zip file using the `Download` button above in this page
2. In the dashboard navigate to `Plugins > Add New`
3. Click on `Upload Plugin`
4. Click `Choose File` then select the zip file you downloaded
5. Click `Install Now`, wait for the installation and click `Activate Plugin`

= Activating reCAPTCHA =

1. Go to `Settings -> Woo reCAPTCHA`
2. Provide the two required keys.
3. Save the setting by clicking the `Save` button. 

1, 2, 3: You're done!
